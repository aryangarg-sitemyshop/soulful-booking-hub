# Project Memory

## Core
Dermatology clinic website "Radiance Skin Clinic". Rose gold primary (#B76E79), cream bg. Playfair Display headings, DM Sans body.
Inspired by bellasoulclinic.com but different brand. Frontend-first approach.
Admin portal at /admin with single admin role. Auth with email/password. Auto-confirm enabled, signup disabled.
DB: appointments, doctors, patients (auto PAT-IDs), profiles, treatment_notes, blog_posts, user_roles.
